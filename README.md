A trivia quiz web application built using HTML, CSS, and JavaScript. Test your knowledge across various topics with questions fetched live from the Open Trivia Database API.

> This project was built to practice JavaScript skills including fetch, async/await, and DOM manipulation.

## Live Link

Check out the quiz in action: [https://quiz.asiradnan.com](https://quiz.asiradnan.com)

## Features

- **Live Questions:** Questions are fetched in real time from the [Open Trivia Database](https://opentdb.com) API.
- **Difficulty Filter:** Choose from Easy, Medium, Hard, or Any Difficulty.
- **Question Count:** Select 5, 10, 15, or 20 questions per session.
- **Timer:** Track how long you take to complete the quiz.
- **Score Summary:** See your final score and time taken at the end.
- **Responsive Design:** Play on any device with a responsive layout.
